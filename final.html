"Flappy Person"

Made using: Khan Academy Computer Science

This is the final project I worked on for CSC 106. It is a take on flappy bird that I made with Ryan Megna. Instead of a bird it utilizes both of our bitmojis as playable sprites.
The code uses an object class to define the pipes and uses an array that pushes in random numbers to determine the y value of the shapes. The game gradually speeds up over time by having the variable "pipeSpeed" added to the x positions, which is constantly being set equal to itself plus a small number.
The game has the choice between three different scenes as well as allowing the choice of bitmoji. The three scenes are "Default" "Sunset" and "Beach", which change the background and the pipe colors.
